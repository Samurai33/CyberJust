## 🚀 Quick Start

### Prerequisites

```bash
Node.js >= 22.15.0
pnpm >= 10.12.4
Git >= 2.34.0
```

### Installation

```bash
# Clone the repository
git clone https://github.com/Samurai33/CyberJust.git
cd CyberJust

# Install dependencies
pnpm install

# Run in development mode
pnpm dev
```

🎉 Open [http://localhost:3000](http://localhost:3000) to see the application running!
